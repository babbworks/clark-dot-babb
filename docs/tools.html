Public tools

Tools anyone can use to support industrial ecosystem work.

Clark builds and publishes tools that serve people working on industrial renewal. Some are software products under the Clarkware name. Others are public utilities that anyone can use without joining Clark or asking permission.

Public utility — Live

Industrial Atlas

An industrial atlas built from public data. Its purpose is to help any person with a hunch or a mature plan discover ideal places to locate their enterprise.

The atlas surfaces regional industrial profiles, workforce data, facility availability, and infrastructure indicators so that site selection decisions can start from evidence rather than guesswork.

Software — In development

Clark IPE

The Integrated Production Environment. A Theia-based workstation application for electronics assembly, firmware, inspection, and evidence capture. Open source, network-native, deployable at any facility.

Read about Clark IPE

Clarkware

Clarkware is the name for the full family of open source industrial software Clark builds. Every tool is designed to offer complete sovereignty, independence, and privacy to its users. Data stays on the operator's own infrastructure. There are no built-in mechanisms or expectations that operators contribute data to Clark or to any network — sharing is always voluntary, selective, and operator-controlled.
