# Clark · Field Report 001

Single-article site for **Field Report 001 — The Lots That Nobody Build On**: ready land, approved plans, real money, nobody building. A report from the western Bayfront, Hamilton, Ontario, and the first act of an industrial activation protocol.

Published at [clark.babb.tel](https://clark.babb.tel) via GitHub Pages.

## Structure

- `index.html`, `styles.css`, `app.js` — the article site (static HTML/CSS, light JS)
  - left rail: section index + working papers (open in an overlay viewer)
  - right rail: sources and reference materials
- `papers/` — HTML editions of the working papers, raw research-stage register
- `project/` — the underlying research and planning records linked from the site
  - `project/papers/` — working papers (markdown source of record)

## Status

Draft v2, 2026-06. Working papers are speculative planning records, not commitments. Corrections are published, not buried.
